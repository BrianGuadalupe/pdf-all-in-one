# PDF Toolkit

**100% client-side PDF editor, merger, splitter & renamer. Nothing leaves your browser.**

## Features

- **Editor** — Cover text, write on top, fill form fields. Inline WYSIWYG preview.
- **Merge** — Combine multiple PDFs into one. Drag to reorder.
- **Split** — Extract specific pages, ranges, or split every page into a ZIP.
- **Rename** — Batch rename PDFs and download as ZIP.

## Usage

Open `index.html` in any modern browser. No install, no server, no dependencies to manage.

## Security

- All processing happens locally in your browser — zero network requests with your data
- Libraries loaded from Cloudflare CDN with **SRI (Subresource Integrity)** verification
- XSS-safe filename handling throughout

## Tech

Single HTML file (~1,400 lines). Built with [pdf-lib](https://pdf-lib.js.org/), [PDF.js](https://mozilla.github.io/pdf.js/), and [JSZip](https://stuk.github.io/jszip/).

# PDF Toolkit (ES)

**Editor, unificador, divisor y renombrador de PDFs. 100% en el navegador. Nada sale de tu equipo.**

## Funcionalidades

- **Editor** — Tapa texto, escribe encima, rellena formularios. Vista previa WYSIWYG.
- **Unir** — Combina varios PDFs en uno. Arrastra para reordenar.
- **Dividir** — Extrae páginas específicas, rangos, o divide cada página en un ZIP.
- **Renombrar** — Renombra PDFs en lote y descarga como ZIP.

## Uso

Abre `index.html` en cualquier navegador moderno. Sin instalación, sin servidor, sin dependencias.

## Seguridad

- Todo el procesamiento ocurre localmente en tu navegador — cero peticiones de red con tus datos
- Librerías cargadas desde Cloudflare CDN con verificación **SRI (Subresource Integrity)**
- Sanitización XSS en todos los nombres de archivo

## Tecnología

Un solo archivo HTML (~1.400 líneas). Construido con [pdf-lib](https://pdf-lib.js.org/), [PDF.js](https://mozilla.github.io/pdf.js/) y [JSZip](https://stuk.github.io/jszip/).
